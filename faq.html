Common Questions

Everything you need to know about the NetPulse engine.

Is NetPulse free to use?

Yes, the core NetPulse reconnaissance suite is free for independent security researchers and community use.

What operating systems are supported?

Currently, NetPulse is optimized for Windows 10/11. Linux and MacOS versions are in the development roadmap for late 2026.

How does OS Fingerprinting work?

NetPulse analyzes the Time-To-Live (TTL) values from target responses. Since different operating systems (Linux, Windows, Cisco) use distinct default TTLs, we can guess the OS with high accuracy.

Is it safe to run 200 threads?

Yes, but it depends on your network stability. High thread counts can trigger IDS/IPS systems or overload low-end routers. We recommend starting at 50 threads and scaling up.
