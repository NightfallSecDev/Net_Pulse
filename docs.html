Scanner Core

The core scanning logic utilizes System.Net.Sockets to perform asynchronous connection attempts. Unlike standard sequential scanners, NetPulse uses a Semaphore-based throttling mechanism to manage concurrent tasks.

var semaphore = new SemaphoreSlim(maxParallel); await semaphore.WaitAsync(ct);

Persistence

Data is stored in a flat JSON structure for portability. The history.json schema captures session metadata, results, and target summaries.

Networking Strategy

NetPulse uses a 'Full-Open' TCP handshake strategy for accuracy. Future updates will include 'Half-Open' SYN scanning for increased stealth.
